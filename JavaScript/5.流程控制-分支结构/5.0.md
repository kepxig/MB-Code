### 2.分支结构

#### if 

```html
if (条件表达式) {
            //执行语句
        }
else if (条件表达式) {
            //执行语句
}
else{
            //执行语句
}
```

##### 三元表达式

```html
//条件表达式？ 表达式1 : 表达式2
//如果条件表达式为真 返回表达式1的值 否则2 

var num=5;
var result= num > 5 ? 'yes' : 'not' ;
console.log(result);//not
```

#### switch

switch (表达式) {
            case value1:
                执行语句1;
                break;

            case value2:
                执行语句2;
                break;
            ...
            default:
                执行最后的语句;
        }
